Каскадні таблиці стилів (CSS) -- мова стилів, що використовується для задання вигляду документів, написаних мовоюHTML або XML (включаючи діалекти XML на кшталт SVG, MathML чи {{Glossary("XHTML")}}). CSS описує, як елементи будуть відрендерені на екрані, папері, у мовленні чи в іншому вигляді.

CSS належить до ключових мов відкритого вебу і стандартизується між веб-бравзерами згідно W3C специфікацій. У минулому розробка різних частин CSS специфікації велася паралельно, що давало змогу випускати свіжі рекомендації у форматі редакцій. Ви могли чути про CSS1, CSS 2.1, CSS3. Між тим, CSS4 так і не стала офіційною редакцією.

Починаючи від CSS3, обсяг специфікації помітно зростав, і прогрес різних модулів CSS почав настільки сильно відрізнятись, що стало більш ефективним розробляти та випускати рекомендації окремо в межах модулів. Замість оформлення редакцій специфікації CSS, W3C тепер періодично робить зліпок найсвіжішого стабільного стану специфікації CSS.

Перші кроки у CSS
CSS (каскадні таблиці стилів) використовуються для стилізації та верстки веб-сторінок - наприклад, для зміни шрифту, кольору, розміру, інтервалів вашого контенту, поділу його на кілька колонок, або додання анімацій та інших декоративних можливостей. Цей модуль дасть м‘який початок вашому шляху до майстерності в CSS у вигляді основ того, як вони працюють, який вигляд має синтаксис, і як можна почати використовувати їх для стилізації HTML.
Структурні елементи CSS

Цей модуль продовжує там, де Перші кроки у CSS зупинились, — тепер, маючи близьке знайомство із мовою та її синтаксисом, і отримавши деякий базовий досвід її використання, пора дещо заглибитись. Цей модуль звертає погляд на каскад та наслідування, всі доступні типи селекторів, одиниці вимірювання, розміри, стилізацію тла та меж, і на багато іншого.

Його метою є надання набору інструментів для написання грамотного CSS та допомога у розумінні всієї суттєвої теорії перед переходом до більш специфічних дисциплін штибу стилізації тексту та CSS верстки.

Стилізація тексту
Після покриття основ мови CSS, наступною темою є стилізація тексту - одна з найбільш поширених справ, котрі доводиться робити на CSS. Тут ми поглянемо на засадничі основи стилізації тексту, включаючи встановлення шрифта, жирності, нахилу, відступу між рядками та між літерами, падіння тіней, та інших особливостей тексту. Ми завершуємо модуль поглядом на встановлення власних шрифтів на вашу сторінку, і стилізацію списків та посилань.
CSS верстка
В цій точці шляху засади CSS уже пройдені, ви уже знаєте, як стилізувати текст і як стилізувати і маніпулювати коробками, в котрих вміщається контент. Тепер пора поглянути на те, як розташувати ці коробки у потрібному місці відносно доступного простору і одне одного. Ми покрили необхідні передумови і можемо заглибитись у CSS верстку, поглянути на різні налаштування екранів, сучасні інструменти верстки штибу flexbox, CSS grid та позиціонування, і на деякі застарілі техніки, про які вам все ж слід знати.

The Шпаргалка CSS прагне зібрати докупи рецепти загальноприйнятих шаблонів верстки, речей, котрі можуть бути потрібними при реалізаці ваших веб-сайтів. Окрім надання коду, котрий можна використовувати як стартову точку власних проєктів, ці рецепти показують різні способи застосування специфікацій верстки, і рішень, котрі можуть бути прийняті розробником.
